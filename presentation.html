What is a CSS preprocessor?

"CSS preprocessors take code written in the preprocessed language and then convert that code into the same old css weâ€™ve been writing for years"

Why use a preprocessor?

  • To fix some of CSS' misgivings and limitations.
  • To make speedy, clean updates to your site.
  • Because of the supporting libraries.
  • They bring traditional operators to CSS. eg. loops, functions.
  • The more they're used, the better they'll get.

Nesting!

Styling a block can be frustrating.


                 section {
                    margin: 10px;
                 }
                 section nav {
                    height: 25px;
                 }
                 section nav a {
                    color: #0982C1;
                 }
                 section nav a:hover {
                    text-decoration: underline;
                 }

In a preprocessor..


                section {
                    margin: 10px;
                    nav {
                        height: 25px;
                        a {
                            color: #0982C1;
                            &:hover {
                                text-decoration: underline;
                            }
                        }
                    }
                }

Variables


                    /*scss*/
                    $color: #efefef;
                    body{ background: $color; }

                    /*less*/
                    @color: #efefef;
                    body{ background: @color; }

                    /*css*/
                    body{
                        background:#efefef;
                    }

Interpolation


    /* -- .scss --  */
    $side: top;
    border-#{$side): 1px solid #000;

    /* -- resulting css -- */
    border-top: 1px solid #000;

    /* -- .less -- */
    @base-url: "http://example.com";
    background-image: url("@{base-url}}/images/bg.png");

    /* -- resulting css -- */
    background-image: url("http://example.com/images/bg.png");
## Mixins (functions!) * Functions that allow the reuse of properties throughout the CSS. * Used to simplify repetitive CSS

A Practical Example


    /*scss*/
    @mixin border-radius($values: 5px) {
        -webkit-border-radius: $values;
        -moz-border-radius: $values;
        border-radius: $values;
    }

    /*less*/
    .border-radius(@values:5px) {
        -webkit-border-radius: @values;
        -moz-border-radius: @values;
        border-radius: @values;
    }

    /*scss*/ .obnoxious-button{ @include border-radius(10px); }
    /*less*/ .obnoxious-button{ .border-radius(10px); }
    /*css*/
    .obnoxious-button{
        -webkit-border-radius: 10px;
        -moz-border-radius: 10px;
        border-radius: 10px;
    }

Inheritence


        .block {
            margin: 10px 5px;
            padding: 2px;
        }
        p {
            @extend .block; /* Inherit styles from '.block' */
            border: 1px solid #EEE;
        }
        ul, ol {
            @extend .block; /* Inherit styles from '.block' */
            color: #333;
            text-transform: uppercase;
        }
                        

        .block, p, ul, ol {
            margin: 10px 5px;
            padding: 2px;
        }
        p {
            border: 1px solid #EEE;
        }
        ul, ol {
            color: #333;
            text-transform: uppercase;
        }

Colour manipulation

  • Apply photoshop-like filters on standard color codes.
  • Rapidly mock up examples of things like hover states
  • Discover color palettes that match your chosen colours

SCSS


    /* returns a color 10% lighter than $color */
    lighten($color, 10%);
    /* returns a color 10% darker than $color */
    darken($color, 10%);
    /* returns a color 10% more saturated than $color */
    saturate($color, 10%);
    /* returns a color 10% less saturated than $color */
    desaturate($color, 10%);
    /* returns grayscale of $color */
    grayscale($color);
    /* returns complement color of $color */
    complement($color);
    /* returns inverted color of $color */
    invert($color);
    /* mix $color1 with $color2 with a weight of 50% */
    mix($color1, $color2, 50%);
so many functions!

LESS


     // return a color which is 10% *lighter* than @color
     lighten(@color, 10%);
     // return a color which is 10% *darker* than @color
     darken(@color, 10%);
     // return a color 10% *more* saturated than @color
     saturate(@color, 10%);
     // return a color 10% *less* saturated than @color
     desaturate(@color, 10%);
     // return a color 10% *less* transparent than @color
     fadein(@color, 10%);
     // return a color 10% *more* transparent than @color
     fadeout(@color, 10%);
     // return @color with 50% transparency
     fade(@color, 50%);
     // return a color with a 10 degree larger in hue than @color
     spin(@color, 10);
     // return a color with a 10 degree smaller hue than @color
     spin(@color, -10);
     // return a mix of @color1 and @color2
     mix(@color1, @color2);
so many functions!
